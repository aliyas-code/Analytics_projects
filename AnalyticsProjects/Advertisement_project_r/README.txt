
Description

(ENG)
The visitors of the Internet site who visited it through advertising were investigated. Information about more than 1000 visits were 
extracted using the Yandex Logs API. Data contains:
-        visitID;
-        isNewUser;
-        clientID;
-        date;
-        bounce;
-        AdType (advertisement type);
-        BannerGroupId;
-        BannerGroupName;
-        BannerId;
-        BannerName;
-        startURL;
-        lastPlatform;
-        Condition (advertising display conditions);
-        deviceCategory;
-        operatingSystem;
-        PageViews;
-        visitDuration;
-        price (price of visit);
-        WatchCount (the number of views that were in this visit).
=====================================================================================================

(RUS)
Исследуются посетители Интернет-сайта, перешедшие на него по рекламе. Была получена информация о более 1000 визитов. Информация о визитах была получена с помощью Yandex Logs API . Информация содержит данные: 
-        visitID (Идентификатор визита);
-        isNewUser (Первый ли визит посетителя);
-        clientID (Идентификатор клиента);
-        date (Дата визита);
-        bounce (Отказность);
-        AdType (Тип рекламы);
-        BannerGroupId (Идентификатор группы баннеров);
-        BannerGroupName (Имя группы баннеров);
-        BannerId (Идентификатор баннера);
-        BannerName (Имя баннера);
-        startURL (Страница входа);
-        lastPlatform (Источник трафика);
-        Condition (Условие показа объявления);
-        deviceCategory (Тип устройства);
-        operatingSystem (Операционная система);
-        PageViews (Количество просмотренных страниц);
-        visitDuration (Продолжительность визита);
-        price (Цена визита);
-        WatchCount (Количество просмотров, которые были в данном визите).
